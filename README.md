# BeeX Robotics Software Engineer Assignment

This repository contains my prepared BeeX robotics software engineering assignment work.

The main handoff package is:

```text
beex_submission.zip
```

The extracted submission folder is also included for readable source review:

```text
submission/
```

## Contents

- `submission/01 Mission Planning/task`: ROS 1 mission planning solution, Docker helper, tests, and documentation.
- `submission/02 Machine Learning/task`: sonar segmentation training code, tests, metrics, and documentation.
- `submission/03 Computer Vision`: numpy-only 3D computer vision solution, tests, answers, and documentation.
- `tasks/`: planning notes, review notes, and lessons captured during development.

Large local-only artifacts are intentionally ignored, including virtual environments, extracted datasets, ROS build folders, exploratory ML outputs, and raw `.pt` checkpoints. The final submission zip is kept because it is the packaged artifact used for handoff.

See `submission/SUBMISSION_README.md` for the recommended review order and task-specific test commands.
